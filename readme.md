# QAcTime v.0.1 #

Простая программа для учёта времени выполняемых действий.

1. Исходный код лежит в папке src.

2. Собрать *.exe с помощью:
- Qt Creator версии не ниже 4.5.1 (как правило, идёт в составе дистрибутива Qt Framework);
- Qt Framework версии не ниже 5.10.1;
- MinGW версии не ниже 53_32 (как правило, идёт в составе дистрибутива Qt Framework).

Для этого:
- открыть файл *.pro в Qt Creator (Файл - Открыть файл или проект или Ctrl + O); все остальные файлы сами подтянутся;
- нажать кнопку "Настроить проект", убедившись, что выбран комплект с MinGW;
- проверить и, если требуется, откорректировать версию exe-файла и год выпуска сборки (файл *.rc);
- выбрать "Выпуск";
- на вкладке проекта выбрать путь для результатов компиляции (/build вместо предлагаемого Qt Creator по умолчанию);
- нажать "Собрать проект" (Сборка - Собрать проект или Ctrl + B);
- выполнить очистку после компиляции:
-- встав на заголовок проекта, вызвать контекстное меню и выбрать пункт "Очистить", или
-- Сборка - Очистить всё.

3. Для установки программы с помощью инсталлятора запустить инсталлятор. После завершения работы инсталлятора запустить программу.
Для удаления программы воспользоваться деинсталлятором, входящим в комплект поставки программы.

4. Для установки программы с помощью zip-файла распаковать содержимое архива в удобное место и запустить программу.
Для удаления программы удалить папку с распакованными файлами.
